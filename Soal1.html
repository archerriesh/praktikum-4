Homepage / Program Studi / Program Sarjana Teknik Informatika

Mahasiswa Program Sarjana Teknik Informatika akan mempelajari konsep ilmu computer science serta implementasi praktis software engineering. Keahlian utama tersebut ditunjang dengan keahlian-keahlian khusus sesuai bidang peminatan:

  1. Game & Mobile Apps
  2. Web & Multimedia
  3. Intelligent Computing
  4. Network Security

Aktivitas Akademik

Fokus utama perkuliahan adalah dalam keahlian pengembangan software (software methods and technologies). Proses pembelajaran seimbang antara konseptual dan praktik di laboratorium, menyiapkan mahasiswa agar siap berkarya di dunia kerja.

Masa Depan dan Karier

Lulusan Program Sarjana Teknik Informatika telah sukses menjadi wirausahawan maupun bekerja di berbagai perusahaan, baik nasional maupun internasional. Lulusan program ini dapat berkarier dalam bidang:

  • Teknopreneur: IT solutions, independent software vendor
  • IT Architect: System Analyst
  • Software Engineer: Mobile Developer, Desktop Developer, Java Developer
  • Database Engineer: Database Developer, Database Administrator
  • System Engineer: Network Engineer, Security Specialist
